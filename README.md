# 🌍 LinguaQuest

> A language learning game built using AI — because learning a new language should feel like an adventure, not a chore.

**[▶ Play LinguaQuest](https://parthkamalakar.github.io/linguaquest)**

---

## 📖 The Story Behind This App

I was using a popular language app — you know the green owl one (Duolingo) — and I thought: *I could build something like this.* I didn't know how to code. But I knew how to describe what I wanted. So I started vibe coding with AI, describing ideas in plain English and watching them come to life.

That first experiment taught me how apps like this are built. Then I went further — and LinguaQuest was born.

---

## ✅ What's In The Game Right Now

- 🌐 **8 Languages** — Spanish, French, German, Japanese, Hindi, Korean, Portuguese, Italian
- 📚 **345 Lessons per language** — 23 topics × 5 lessons across 3 skill sections (Rookie → Explorer → Champion)
- 🔥 **Streaks & XP** — earn points, level up, keep your streak alive
- ⭐ **Unit Stars** — get up to 3 stars on every level test
- 🏅 **Achievements & Badges** — unlock milestones as you progress
- 📅 **Daily Challenges** — 20 days of bonus challenges once you finish the course
- 🗺️ **Quests** — daily mini-goals to keep you motivated
- 🏆 **Leaderboard** — compete and see how you rank
- 🗣️ **Speaking Exercises** — practice pronunciation out loud
- 🔊 **Sound & Text-to-Speech** — hear words spoken in the real language
- 🐦 **Custom Characters** — meet Hoot, Falstaff, Lily, Bea, Junior, Eddy, Zari, and Oscar
